# Shantanu Kumar - Structural Bioinformatics Portfolio

A premium, modern React portfolio for a Structural Bioinformatician, featuring glassmorphism design, custom animations, and integrated CV details.

## 🚀 Features

- **Modern Tech Stack**: Built with React (Vite), Framer Motion for animations, and Lucide React for icons.
- **Glassmorphism Design**: Sleek dark theme with semi-transparent panels and vibrant accents.
- **Interactive Background**: Animated SVG background inspired by biological structures.
- **Project Showcases**: Detailed cards with protein simulation videos and research tags.
- **Expertise Grid**: Comprehensive list of tools and categories from the CV.
- **Responsive Layout**: Designed to look great on all devices.
- **Interactive Animations**: Custom border-drawing hover effects on all cards.

## 🛠️ Prerequisites

Before you begin, ensure you have the following installed:
- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) (usually comes with Node.js)

## 📦 Installation & Local Development

1. **Navigate to the project directory**:
   ```bash
   cd portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **View the website**:
   Open [http://localhost:5173](http://localhost:5173) in your browser.

## 🏗️ Production Build

To create a production-ready build of your portfolio:

1. **Build the project**:
   ```bash
   npm run build
   ```

2. **Preview the build**:
   ```bash
   npm run preview
   ```

The output files will be in the `dist` folder, which can be deployed to platforms like GitHub Pages, Vercel, or Netlify.

## 📂 Project Structure

- `src/components/`: Individual sections of the portfolio (Hero, About, Projects, etc.).
- `src/App.jsx`: Main application logic and section assembly.
- `src/index.css`: Global styles, CSS variables, and glassmorphism definitions.
- `public/`: Assets like CV PDF, thesis, and protein simulation videos.
